1. What is Semantic HTML? 
HTML that uses descriptive element names that can easily be read/understood by people. For example: <header> or <section> or <body> etc.
    2. What is HTML used for? 
    To adjust the layout of a web page.
    3. What is an attribute and where do we put it? 
    Inside an opening element tag.  They control an elements behavior.  For example, they can reference and image from the web or link to a webpage.
    4. What is the h1 tag used for? How many times should I use it on a page?
    It should only be used once per page and is for the most important part of a page.  Usually a title or header.  It is the largest font size of all <h> elements.
    5. Name two tags that have required attributes
    The <a> or anchor tag and the <img> or image tag.

    37. Walk me through Lambda's git flow. 
    First, fork the repository from github.  Then clone the repo.  Open it in git using the git clone "repo url" command.
    Next do git checkout -b 'your-name' to create a branch.  Edit the file as needed.  git commit to save changes.  git add . to add 
    any new files to your branch.  git push to send a copy of your branch to github for review.  Create a pull request on GitHub.

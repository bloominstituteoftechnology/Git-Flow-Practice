
    1. What is Semantic HTML? 
Semantic HTLM is a type of HTML code that focuces on labeling and adding meaning for both the coder and the browser itself.  

    2. What is HTML used for? 
HTML is used for creating electronic documents, AKA webpages, to be displayed in any web browser.

    3. What is an attribute and where do we put it? 
Attributes are special lines of code used to modify HTML elements used inside the opening tag.

    4. What is the h1 tag used for? How many times should I use it on a page?
H1 is used for headings/titles, most specifically the main heading or title and shouls only be used once.

    5. Name two tags that have required attributes

    6. What do we put in the head of our HTML document? 
Title, links to CSS (style), and any meta data are placed in the head of an HTML document.

    7. What is an id? 
Id in HTML is a global attribute used to specify a unique element.

    8. What elements can I add an id to? 
<h1>___ </h> and <p>___</p> are elements that can have an added id.

    9. How many times can I use the same id on a page? 
You can only use an id once.

    10. What is a class? 
Class in HTML is a global attribute used to give a name to a specified section in your code in order to be reffereced later by CSS and JS.

    11. What elements can I add a class to? 
<p>___</p>, <div>__</div>, and <section>__</section> are a few elements I have been using with class.

    12. How many times can I use the same class on a page? 
You can only use the same class once per page however, you can create as many classes as you want as long as they have their own specified names.  

    13. How do I get my link to open in a new tab?
You can get links to open in a new tab or window by adding "target="_blank" after your "href=URLHERE".

    14. What is the alt attribute in the image tag used for? 
Alt attributes in the image tag are used to add specified text in the event the image cannot be displayed.

    15. How do I reference an id?
To reference an id in CSS you would use "#" plus the id name with no space.

    16. What is the difference between a section and a div
A section is an element that represents a grouping of content where a div represents its children.

    17. What is CSS used for? 
CSS is used for styling HTML text and images to be more appealing and accessible to the user.

    18. How to we select an element? Example - every h2 on the page
To select element h2 in CSS, you would simply add h2 in its own line. No need for a "." like you would if you were selecting a class.

    19. What is the difference between a class and an id? - Give me an example of when I might use each one.
Class can be used to identify multiple elements where as id can only identify one. Id would be used if you have a URL with an added #. When clicked the page will automatically scroll to that specified element. Class would be used when making changes to multiple items in CSS.

    20. How do we select classes in CSS?
To select a class in CSS simply place a "." in front of the name you gave it in HTML.

    22. What is a parent child selector? When would this be useful? 
Parent child selector is used to select all elements that are a direct child of a specified element. This tool would be useful in selecting a list of items to add the same attribute to.

    23. How do you select all links within a div with the class of sidebar?
To select all the links in the class of sidebar you would use ".sidebar" in CSS.

    24. What is a pseudo selector?
A pseudo selector is a tool that selects elements that are in a specific state. One example of which would be hover state.

    25. What do we use the change the spacing between lines?

    26. What do we use to change the spacing between letters?
